# AI Energy Management Agent for Educational Institutions and Workplaces

## AICTE Generative AI & Agentic System Engineering Internship Project

### Project Description
This project uses Google Gemini AI to analyze institutional energy consumption data and generate intelligent energy optimization recommendations.

### Features
- Institutional Energy Dataset
- Energy Consumption Visualization
- AI-Based Energy Analysis
- High Energy Consumption Detection
- Carbon Emission Estimation
- Energy Saving Recommendations
- Smart Monitoring Suggestions
- SDG 13 (Climate Action) Support
- AI-Generated Institutional Energy Management Report

### Technologies Used
- Python
- Google Colab
- Google Gemini API (google-genai)
- Pandas
- Matplotlib

### Project Output
- Energy Consumption Charts
- Carbon Emission Analysis
- AI Optimization Report
- Institutional Energy Management Report
